# Task 1-AI Website Copy Generator
# AI Website Copy Generator for Local Businesses

## Overview

Many local businesses struggle to convert website visitors into customers because:

- Their website copy is unclear
- The value proposition is weak
- Calls-to-action are confusing
- Content feels generic or outdated

Professional copywriters can be expensive, and many business owners don’t know what to write.

This project demonstrates how **Prompt Engineering + AI tools** can generate **conversion-focused website copy** for local businesses.

The system uses **structured prompts** to produce website-ready content including:

- Homepage copy
- Services page content
- Call-to-action sections
- Tone adaptation for different business types

This approach can be used by:

- Web designers
- Freelancers
- Agencies
- Local businesses

# Prompt Framework

The system uses **reusable structured prompts** that generate:

1️⃣ Homepage Copy  
2️⃣ Service Descriptions  
3️⃣ Call-To-Action Sections  
4️⃣ Tone Adaptation

Each prompt is designed to produce **clear, persuasive, conversion-focused website content**.

---

# Prompt Design Logic

The prompts are structured to guide the AI to:

- Understand the business
- Identify the customer pain points
- Highlight benefits instead of features
- Create strong CTAs
- Maintain a consistent tone

Each prompt includes:

Input Variables:
- Business name
- Business type
- Location
- Target customers
- Unique selling points
- Tone

This allows the prompts to be reused for **any local business** such as:

- Salons
- Cafes
- Clinics
- Coaching institutes
- Agencies
